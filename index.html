Cabshare

About: Cabshare is an application created to share cab expenses for longer travels. A example route would be from a college to airport and vice-versa

How It Works: Cabshare matches people traveling from nearby origin to nearby destination points. The application takes the origin and destination points of two people and finds the sum of distance between origin points and destination points. If the sum is below a certain limit the ride is matched.

For a demo please visit Cabshare

Rhythm Tapper

About: Rhythm Tapper is an application that lets you tap to a given song. The application currently has only song to which one can tap.

How It Works: Rhythm Tapper contains a master array which contains the timestamps at which one has to tap. The application records the timestamps of your taps and matches with the master array. Depending how accurately you tap a colour between red and green will be displayed on the page.
